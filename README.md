# AI-Powered Supply Chain Optimizer

This project is an AI-based system that helps businesses manage inventory efficiently using demand forecasting and risk analysis.



## Features
- Demand Forecasting using Machine Learning
- Inventory Risk Score Calculation
- Reorder Quantity Recommendation
- Dashboard Visualization

## Tech Stack
- Python (Flask)
- Machine Learning
- HTML, CSS, JavaScript
- SQLite

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py
